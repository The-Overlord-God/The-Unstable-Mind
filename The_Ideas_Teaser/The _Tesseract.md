# the Tesseract

The tesseract is a cube made out of LEDs and transistors. It kinda looks like the tesseract from Marvel when it’s covered with diffused sheets. It’s best for a lamp or decorative projects and gives a really cool glowing effect.

# How it works

So basically I use a simple 3D plane system to operate each LED, like coordinates (1, 3, 4). Each LED is placed in a 3D space, so you can control them layer by layer.

All the LED anodes are connected in parallel, and the cathodes are connected in series along the axes. Each axis has like 10 wires connected (or something like that), so it forms a grid in 3D.

The transistors are connected in series with the LEDs, so we can control the current going through them. By controlling the current in different rows and layers, we can turn on specific LEDs and create patterns all over the cube.

It’s not super complicated, just a bit messy wiring-wise, but once it works it looks really awesome
